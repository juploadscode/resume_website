Media Background Replacement

Placeholder

The idea for this project spawned when watching a Twitch.tv stream. A broadcaster I was watching made a comment about how frustrating it is to always have green screen equipment behind them all the time, giving me the idea to make software to do the job for them. To create this project I used Google's Mediapipe library for selfie segmentation, which worked hand in hand with OpenCV to create the background replacement aspect of the software.


Placeholder

The GUI of this application utilizes PyQt5 and allows users to Browse files to upload for background replacement. The program currently accepts .PPT, .MP4, .PNG, and .JPG file formats. Eventually I plan on adding more file types including PDF, Word documents, and various photo/video formats that are currently not included.
