eBay Buy Bot

Placeholder

I created this program as a business tool for reselling items on eBay. The application takes two parameters, an eBay link and a max bid amount. Once the GUI has both values entered, the start button is enabled and allows the user to start the bot. The program waits until the very last second of an auction and then places a bid as the max bid variable value.


Placeholder

To create this program I used an open source library called selenium. Selenium allowed me to manipulate my web browser when needed. I also used PyQt5 for the GUI of the application, allowing me to display all relevant information about the auction on a clean interface. I used threading to create a timer to run against the real auction timer, taking the displayed time minus a few seconds to auto bid on the item at the very last second of the auction.
