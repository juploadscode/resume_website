Pokemon Database Project

Placeholder

I created a simple SQL database to store Pokemon card prices. The database has various tables for all of the different card sets. To upload data to the database itself, I wrote a simple Python script to web scrape price data from reputable online sources. After recording and observing data for a few months, I was able to figure out which cards are worth buying from an investment standpoint. Analysis also showed me which cards I can buy cheap to resell for profits. Various queries provide me quick and easy business analysis.
