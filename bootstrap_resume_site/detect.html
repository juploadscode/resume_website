Custom Object Detection

Placeholder

To learn more about machine learning, I trained various object detection models. Tutorials on YouTube and Github showed me how I can train custom datasets with neural networks. These resources introduced me to Yolo or You Only Look Once, a object detection approach that works with a convolutional neural network called Darknet. I have a commented tutorial on how I trained my datasets posted on my Github.


Placeholder

The first dataset I trained was four wheel drive vehicles. I ended up labeling over 1,000 images of various vehicles in dramatically different settings, providing extremely accurate detection results after training multiple epochs. I also trained a lighter model in yolov3 to detect pokemon cards. Once I had an accurate, working model, I incorporated the dataset into a Python program to detect specific cards in large card lots posted on eBay and other online marketplaces.
